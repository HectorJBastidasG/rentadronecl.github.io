Rentadrone Developers

This is an initiative started by Rentadrone.cl to establish an inclusive community of AI researchers for the development and exchange of best practices for AI in thermographic imaging.

A part of our workload is dedicated to open source and side projects. You can spot team members in user groups or as conference speakers as well.

Photovoltaic Fault Detector

Based in model detector YOLOv.3, this repository contains four detector model with their weights and the explanation of how to use these models.

Precision Agriculture

Set of Machine Learning Algorithms developed with the aim of determining health states of different types of crops.

Web-API

API specification for deploy the detection models, this repository contains a performant, production-ready reference implementation.

UI

Production-ready reference implementation of an administration tool, which gives you easy access to the EXIF meta-data of an image, generate KMZ and more.
